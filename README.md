# Groo Issues

This is the public issue tracker for all Groo products. Use this repository to report bugs, request features, or ask questions.

## How to Submit an Issue

1. Click the **Issues** tab above
2. Click **New Issue**
3. Choose a template:
   - **Bug Report** - Something isn't working
   - **Feature Request** - Suggest an improvement
4. Fill out the form and submit

## Security Vulnerabilities

**Do not report security vulnerabilities publicly.**

Please email [security@groo.dev](mailto:security@groo.dev) with details about the vulnerability. We will respond within 48 hours and work with you to address the issue.
